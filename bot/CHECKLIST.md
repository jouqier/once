# Чек-лист после деплоя

Отметьте каждый пункт после проверки:

## 1. Heroku

- [ ] Бот задеплоен на Heroku
- [ ] Dyno запущен (`heroku ps`)
- [ ] Логи показывают "✅ Бот запущен успешно"
- [ ] Все переменные окружения установлены (`heroku config`)

## 2. BotFather

- [ ] Inline Mode включен
- [ ] Web App создан
- [ ] Короткое имя приложения = `app` (или другое из `.env`)

## 3. Тесты бота

- [ ] `/start` - бот отвечает приветствием
- [ ] `https://t.me/onceappbot?start=movie_550` - показывает Fight Club с постером
- [ ] `@onceappbot share_movie_550` - показывает карточку для шаринга
- [ ] Карточка отправляется в чат с кнопкой

## 4. Тесты Mini App

- [ ] Приложение пересобрано (`npm run build`)
- [ ] Кнопка "Поделиться" работает
- [ ] "Поделиться с превью" открывает диалог Telegram
- [ ] Карточка с постером отправляется
- [ ] "Скопировать ссылку" копирует Direct Link
- [ ] Direct Link открывает Mini App с нужным фильмом

## 5. Проверка ссылок

- [ ] Формат Direct Link: `https://t.me/onceappbot/app?startapp=movie_550`
- [ ] Ссылка открывает Mini App автоматически
- [ ] Параметры передаются правильно (проверьте в консоли)

## 6. Мониторинг (опционально)

- [ ] UptimeRobot настроен
- [ ] Health check работает
- [ ] Алерты настроены

## Если что-то не работает

### Бот не отвечает
```bash
heroku restart --app your-app-name
heroku logs --tail --app your-app-name
```

### Inline mode не работает
1. Проверьте BotFather (Inline Mode включен?)
2. Подождите 2-3 минуты
3. Проверьте логи бота

### Direct Link не работает
1. Проверьте `APP_SHORT_NAME` в Heroku config
2. Убедитесь, что совпадает с BotFather
3. Пересоберите приложение
