feat: разделение хранилищ фильмов и сериалов (v1.2.0)

Проведен рефакторинг структуры хранения данных для четкого разделения
фильмов и сериалов в отдельные хранилища.

## Основные изменения

### Структура данных
- Фильмы теперь хранятся только в `movies` (want, watched)
- Сериалы хранятся в `tvShows` (want, watching, watched)
- Удалено `movies.watching` (фильмы не могут быть в watching)

### Новые методы API
- getTVShowState() - получение состояния сериала
- addTVShowToWant/Watching/Watched() - добавление сериала
- removeTVShowFromWant/Watching/Watched() - удаление сериала
- getTVShowWantList/WatchingList/WatchedList() - получение списков

### Миграция данных
- Автоматическая миграция 1.1 → 1.2
- Перенос сериалов из movies в tvShows
- Валидация и очистка данных
- Удаление дубликатов

### Валидация
- Защита от добавления фильмов в watching
- Защита от смешивания типов контента
- Автоматическая проверка media_type
