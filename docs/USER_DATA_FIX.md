# Исправление проблем с данными пользователя

## Если кнопки работают некорректно

После обновления приложения у некоторых пользователей могут возникнуть проблемы с работой кнопок (например, отметка серии не меняет основные кнопки).

### Автоматическое исправление

Приложение **автоматически исправляет** эти проблемы при загрузке. Просто:

1. Перезагрузите страницу (F5 или Cmd+R)
2. Все данные будут автоматически обновлены
3. Ваши списки фильмов и сериалов сохранятся

### Ручное исправление (если автоматическое не помогло)

Если проблемы остались:

1. Откройте консоль браузера (F12)
2. Введите команду:
   ```javascript
   dataRepair.diagnose()
   ```
3. Если обнаружены проблемы, введите:
   ```javascript
   dataRepair.repair()
   ```
4. Перезагрузите страницу

### Что происходит при исправлении?

- ✅ Все ваши данные сохраняются
- ✅ Автоматически создается резервная копия
- ✅ Структура данных обновляется до новой версии
- ✅ Исправляются все некорректные записи

### Нужна помощь?

Если проблемы не исчезли, в консоли браузера введите:
```javascript
dataRepair.help()
```

Это покажет все доступные команды для диагностики и исправления данных.

---

**Важно:** Ваши данные (списки фильмов, оценки, просмотренные эпизоды) никогда не удаляются при исправлении. Они только конвертируются в новый формат.
